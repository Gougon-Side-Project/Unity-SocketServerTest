# Unity-SocketServerTest
**說明**<br>
* 達成Unity以及Android間的互相溝通
* 此為Unity 伺服器端 / 客戶端

**主要功能**<br>
* 藉由socket讓彼此間互相共享資訊
* 實際動作
  * 輸入IP及PORT後，按下connect鈕可以讓client連接到server
  * server會偵測有誰連接
  * 可以藉由send來送出客戶端訊息
  * server會廣播訊息給所有clients
